# PalindromeCheckerApp

## Use Case 2

### Hardcoded Palindrome Validation

This use case checks whether a predefined (hardcoded) string is a palindrome.

### Flow:
1. Program starts
2. Hardcoded string is stored
3. Characters are compared from both ends
4. Result is displayed
5. Program exits

### Concepts Used:
- Class
- main() method
- String
- if-else condition
- for loop
- System.out.println()
